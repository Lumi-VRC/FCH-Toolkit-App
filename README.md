# The FCH-Toolkit App
The **standalone** solution for moderation assistance, with optional integration with [https://fch-toolkit.com]

Simply navigate to Releases and download the latest EXE.
Virustotal scans always included with each installer!

## Features:

## - Instance Monitor -
- Shows users in instances with applicable data.
- Allows you to control local notifications and modify notes for users in your instance.
Notifications: - Play a noise when the user joins your instance!
Notes: Add a permanent note to their account!

## - Database -
- Displays all users that you have added notifications to, and their notes.
- Allows you to manually add an entry for people not currently in your instance.

## - Log Explorer -
- Allow you to search and filter the last 1000 lines of your own VRChat log!

## - World Moderation -
- Tracks warns/kicks/bans using Reimajos Moderation Panel (Currently in beta, NOT released yet. Soon!)

## - Group Watchlist Integration (FCH-Toolkit.com)
The Watchlist is a feature of the main website, allowing a group to build a centralized database of users.
- Get your access key from the management page for your group, if your group owner has it enabled.
<img width="1508" height="363" alt="image" src="https://github.com/user-attachments/assets/9a19d23a-a41b-49e6-9d6a-80c40687495c" />
- Add your group by inputting it into the login page.
<img width="1497" height="1165" alt="image" src="https://github.com/user-attachments/assets/79414877-b808-4ef2-bb6f-44d472322b30" />
- Group Watchlisted users will glow in your Instance Monitor. Click them to show their notes + moderation history!
- - Yellow: Notifs Off
- - Red: Notifs On
